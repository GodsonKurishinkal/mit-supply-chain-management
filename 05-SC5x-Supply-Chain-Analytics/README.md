# SC5x: Supply Chain Analytics

## Course Overview
This course covers advanced analytics techniques applied to supply chain management, including predictive analytics, machine learning, and optimization algorithms.

## Topics Covered

### Week 1-3: Descriptive Analytics
- Data visualization
- Statistical analysis
- Performance metrics and KPIs

### Week 4-6: Predictive Analytics
- Machine learning for demand forecasting
- Classification models
- Regression techniques

### Week 7-9: Prescriptive Analytics
- Optimization algorithms
- Simulation-based analytics
- Decision support systems

### Week 10-13: Advanced Topics
- Real-time analytics
- Big data in supply chains
- Analytics implementation

## Learning Objectives
- [ ] Apply descriptive analytics to supply chain data
- [ ] Build predictive models for forecasting
- [ ] Develop prescriptive analytics solutions
- [ ] Implement end-to-end analytics pipelines

## Key Techniques
- Time series forecasting (ARIMA, Prophet)
- Machine learning (Random Forest, XGBoost)
- Linear and integer programming
- Monte Carlo simulation
- Deep learning applications
